#### Design A twitter

###### functional requirements
1. user should able to create profile and add profile picture
2. user should able to write tweet and post it
3. you will see feed from your followers 
4. we can see trending news
5. user should able to reply, comment and like a post

above rerquirements are high level requirements there may be other additional feature that we are not aware of them.


##### Non functional requirements
1. system should able to handle millions of users
2. system should be up 99.99%
3. we need eventual consistency to increase latency.
